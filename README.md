# 📋 Kanban Board — Drag and Drop

Projeto desenvolvido para a Prova P1 da disciplina de Desenvolvimento Web.
Um quadro Kanban interativo com funcionalidade de arrastar e soltar cards entre colunas.

---

##  Funcionalidades

- ✅ Arrastar cards entre as colunas **A Fazer**, **Fazendo** e **Concluído**
- ✅ Criar novos cards
- ✅ Editar o texto do card com duplo clique
- ✅ Design responsivo (funciona no celular e no computador)

---

## Tecnologias

- HTML5 (API nativa de Drag and Drop)
- CSS3 (Flexbox e Grid)
- JavaScript puro

>  Nenhum framework foi utilizado neste projeto.

---

##  Estrutura do projeto
